# Examples

This directory contains a deliberate demo output package, not a live client deliverable.

Included sample:
- `ridge-wallet-preview/` — a complete example page package with strategy, copy, visuals manifest, built HTML, metadata, and QA report.
- `stealads-preview/` — a screenshot-led SaaS example with concept-support visuals and draft-only QA notes.

Use it to inspect the artifact contract and design standard. Do not copy any placeholder proof claims into production without source verification.
